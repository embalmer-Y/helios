# Simulated operator dialogue for the CLI real-LLM long pre-run (scripts/cli_dialogue_prerun.py).
# One operator line per non-comment line. Lines are fed through the CLI channel driver as real,
# varying external afferents (so 03 novelty/threat/reward/social vary and 04/05 affect responds),
# while a real LLM does the 11 thinking. Blank lines and lines starting with '#' are ignored.
#
# The mix is deliberately varied (greeting, factual, emotional, social, self-referential,
# challenging, tool-ish, follow-up) so the long run is NOT a frozen-input run.
