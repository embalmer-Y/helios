## 1. How To Read This Document

This is a qualitative distance assessment, not a fake-precision percentage report.

The reference target is the v2.0.0 release standard in `ARCHITECTURE_PHILOSOPHY.zh-CN.md`, especially:

1. architecture completeness,
2. single-runtime closed-loop execution,
3. multi-tick subjective continuity,
4. controlled outward externalization,
5. governed identity evolution,
6. read-only evaluation and falsifiability.

## 2. Distance Scale

| Distance level | Meaning |
| --- | --- |
| `low` | the owner is near final-goal adequacy; remaining work is mainly policy deepening, stronger integration, or better evidence |
| `medium` | the owner has a real baseline, but final-goal closure still needs meaningful depth or downstream consequence |
| `high` | the owner exists or is partially mature, but still leaves a large final-goal gap in the v2.0.0 standard |
| `very_high` | the owner or documentation slice is too early for final-goal claims; substantial implementation or closure work remains |

## 4. What This Matrix Says In Practice

1. `01-02` are no longer the main release blockers.
2. `03-12` mostly sit in the middle band: real owners exist, but their final-goal adequacy depends on stronger downstream consequence and longer-horizon integration.
3. `13-18` carry the heaviest remaining runtime distance because they determine whether internal state really becomes governed action, continuity, and falsifiable subjectivity.
4. `17` and `18` are the strongest near-term bottlenecks to the final-goal standard.
5. `19-20` are not runtime blockers by themselves, but they are release-truth blockers if runtime reality and documentation drift apart.

## 5. Near-Term Priority Reading

If the goal is to reduce distance to v2.0.0 fastest, the current order is:

1. `17` evaluation fidelity and diagnostic provenance.
2. `18` subjective autonomy and proactive evolution.
3. `13` plus `16` execution-closure path.
4. `14` plus `15` for longer-horizon governed continuity.
5. `19-20` as truth-maintenance companions during every major owner wave.
