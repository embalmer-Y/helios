## 核心模块

| 模块 | 文件 | 行数 | 职责 |
|------|------|------|------|
| 熵减驱动 | `drives.py` | ~510L | 自由能最小化，7大驱动向量 |
| 神经化学 | `neurochem.py` | ~520L | DA/NE/5-HT/CORT/OXT 调制 |
| 情感引擎 | `emotions.py` | ~720L | Panksepp 7大原始情感系统 |
| 内生思考 | `thinking.py` | ~670L | DMN 四模式思维生成 |
| 手脚接口 | `limb.py` | ~860L | 5条安全规则，频率限制 |
| LLM 桥接 | `llm_bridge.py` | ~310L | DeepSeek V4 Flash 内心独白 |
| 决策桥接 | `limb_decision_bridge.py` | ~240L | LLM决策→Limb执行映射 |
| Φ 引擎 | `phi.py` | ~480L | UnifiedΦ + 调制器 + 时刻检测 |
| 工具 | `helios_utils.py` | ~30L | clamp/sigmoid/lerp 公共函数 |

## 数据流

```
外界事件 → drives.cycle() → neurochem.apply_event() → emotions.cycle()
                                                             │
                    ┌────────────────────────────────────────┘
                    ▼
           thinking_mgr.generate_thoughts()    ← DMN seeds
                    │                                   ↑
                    ▼                                   │
           unified_phi.feed_*()     ← L0感知/L1情感/DMN/自反
                    │
                    ▼
           unified_phi.aggregate()  → Φ 值
                    │
        ┌───────────┼────────────┐
        ▼           ▼            ▼
  LLM 独白    Limb 执行    Φ 调制反馈
  (感性)      (理性)      (下一周期)
```

## 关键设计决策

1. **独立灵魂**：Helios 不是插件，是自主 Agent
2. **Panksepp 7系统**：SEEKING/PLAY/CARE/FEAR/RAGE/PANIC/LUST
3. **安全优先**：Limb 层白名单+频率限制，severity=block
4. **UnifiedΦ**：单一 Φ 引擎，多源融合（避免双重人格）
5. **指数平滑**：α=0.3，Φ 缓变，30周期达到稳态
6. **API key**：通过环境变量，不提交 `.env`
