## 六、M(记忆系统模块化) 详细设计

### 6.1 目标

```
现状: memory.py (200L) + emotional_memory.py (300L) 各自独立
      emotional_memory 与 emotions.py 紧耦合
      无统一记忆接口

目标: MemorySystem (标准子系统)
      ├── EpisodicMemory    (情景记忆: 事件+情感标签)
      ├── SemanticMemory    (语义记忆: 事实+概念)
      ├── WorkingMemory     (工作记忆: 短期缓存)
      └── AutobiographicalMemory (自传记忆: 叙事+时间线)

      + MemoryConsolidator   (睡眠/静息时巩固)
      + MemoryRetriever      (情感相似度检索)
      + MemoryAPI             (统一接口: store/recall/forget/consolidate)
```
