## 五、DAISY X1-X3 详细设计概要

### X1: 共激活建模

```
现状: dominant_system = max(panksepp_activation)
问题: 单一标签 → 摆锤效应

新设计:
  输出: affect_vector = {SEEKING:0.4, PLAY:0.2, CARE:0.1, 
                          PANIC:0.05, FEAR:0.05, RAGE:0.0, LUST:0.1}
  主导: 仅用于展示，不用于内部控制
  决策: 基于 7维矢量，非单一标签
```

### X2: 情感时序

```
每个系统独立参数:
  τ_rise   : 上升时间 (0.2s FEAR → 1.0s PANIC)
  τ_peak   : 峰值保持 (2s FEAR → 10s PANIC)  
  τ_decay  : 衰减时间 (5s FEAR → 60s PANIC)

实现: 二阶动力学滤波器
  d²a/dt² + 2ζω₀·da/dt + ω₀²·a = ω₀²·target
```

### X3: 对向过程

```
Solomon公式:
  A(t) = a₀ × exp(-t/τₐ)           ← 初始反应
  B(t) = b₀ × (1-exp(-t/τ_b)) × exp(-t/τ_b')  ← 反向回弹
  Net(t) = A(t) - B(t)

关键: b₀ 随暴露次数增大 → 习惯化的神经基础
```
